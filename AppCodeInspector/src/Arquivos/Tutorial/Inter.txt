• Regras para o nível de dificuldade Intermediário:
	1. O jogo consiste em apresentar trechos de artefatos (ex: código ou requisitos) que possuam algum defeito.
	2. O usuário tem que selecionar o trecho que possua algum defeito e indicar qual o tipo do defeito.
	3. Quando o usuário avançar para uma próxima pergunta, não poderá voltar para responder as perguntas anteriores.

• Modo de jogar:
	1. O jogador deverá procurar e selecionar o trecho do artefato que representa um possível defeito.
	2. O jogo exibirá uma lista com alguns tipos de defeitos.
	3. O jogador deverá classificar o defeito selecionando um dos tipos de defeito.
	4. O usuário deverá selecionar a opção para ir para a próxima questão.
	5. Os passos 1 a 4 devem ser repetidos até que não haja mais questões.
